## 什么是插值函数？
插值是一种通过已知离散数据点来估算未知点数值的方法。简单说，就是“用已知点去推算两点之间某点的值”。
这个概念是插值的普遍概念，我们所使用的Lerp函数只指的是线性插值，
### 线性插值
**原理**：假设两点之间是一条直线。
**公式**：  
在点 $(x_0, y_0)$ 和 $(x_1, y_1)$ 之间，求 x 处的 y：
$$y = y_0 + (x - x_0) \cdot \frac{y_1 - y_0}{x_1 - x_0}$$
**特点**：快速、简单，但不够平滑。

## Lerp函数介绍
对于一维情况：
$$Lerp(a,b,t)=a+(b−a)×tLerp(a,b,t)=a+(b−a)×t$$

其中：
- a：起始值
- b：结束值
- t：插值因子，通常范围 [0,1]
t的作用是返回两点中间的百分比位置，比如当t=0.25，lerp就返回a到b中间25%的值。

## **视差遮蔽映射**
{% codeblock [title] [lang:Shaderlap] [url] [link text] [additional options] %}
float2 ParallaxOcclusionMapping(float2 uv, float3 viewDirTS, Texture2D heightMap, SamplerState heightSampler, float scale)
{
    // 层数配置
    const int minLayers = 8;
    const int maxLayers = 32;
    
    // 动态计算层数
    float numLayers = lerp((float)maxLayers, (float)minLayers, abs(dot(float3(0, 0, 1), viewDirTS)));
    
    // 计算步长
    float layerDepth = 1.0 / numLayers;
    float currentLayerDepth = 0.0;
    
    // 纹理坐标偏移量
    float2 deltaUV = viewDirTS.xy * scale / numLayers;
    float2 currentUV = uv;
    
    // 获取当前高度
    float currentHeight = heightMap.SampleLevel(heightSampler, currentUV, 0).r;
    
    // 分层搜索
    while (currentLayerDepth < currentHeight)
    {
        currentUV -= deltaUV;
        currentHeight = heightMap.SampleLevel(heightSampler, currentUV, 0).r;
        currentLayerDepth += layerDepth;
    }
    
    // 精确交点计算
    float prevDepth = currentLayerDepth - layerDepth;
    float prevHeight = heightMap.SampleLevel(heightSampler, currentUV + deltaUV, 0).r;
    
    float weight = (currentLayerDepth - currentHeight) / 
                   ((currentLayerDepth - currentHeight) - (prevDepth - prevHeight));
    
    return currentUV + deltaUV * weight;
}
